# Changelog

All notable changes to the Directed Systems manifesto will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## Release Schedule

New versions are released every 6 months:
- **June** releases (v0.2, v0.4, v0.6...)
- **December** releases (v0.1, v0.3, v0.5...)

Version 1.0 will be released when the core thesis is validated by market reality.

## [0.1.0] - 2025-12-24

- Core framework: The Five Components (Directing Agent, Operational Framework, Taskifier, Operator, Feedback Loop)
- The Three Powers model (Framework, Execution, Feedback)
- The Transition Window concept
- Emerging roles vocabulary (Directed Operators, Framework Architects, Exception Debuggers)
- The Principle of Legibility
- Website launch at [directed.systems](https://directed.systems)

### Foundation
- Established vocabulary for the inversion of cognitive hierarchy
- Defined the "High-Fidelity Biological Interface" concept
- Articulated the path of resistance vs. path of adaptation
